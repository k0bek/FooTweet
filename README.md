
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/k0bek/FooTweet.git
   ```
2. Enter the folder

   ```sh
   cd FooTweet
   ```

3. Install NPM packages
   ```sh
   npm install
   ```

## Features

- logging in
- creating posts
- commenting on posts
- liking posts
- changing username, bio, profile picture, cover photo
- following users
- checking the number of followers and followings
